flume - read status from flume water sensor and record to influxdb

* Install
bundle install
* Scan
Usage:
  flume.rb record-status

Options:
      [--log], [--no-log]          # log output to ~/.log/.nest.log
                                   # Default: true
  -v, [--verbose], [--no-verbose]  # increase verbosity

Record current usage data to database
